# 🩺 Diabetes Prediction using Machine Learning

A machine learning project that predicts whether a patient has diabetes based on medical attributes using **Support Vector Machine (SVM)**.

---

## 🎯 Project Overview

This project builds a classification model to predict diabetes in patients using the **Pima Indians Diabetes Database**. The model uses **Support Vector Machine (SVM)** with feature scaling to achieve optimal performance.

---

## 🚀 Key Features

- Data preprocessing
- Feature scaling using **StandardScaler**
- Classification using **Support Vector Machine (SVM)**
- Model evaluation with **accuracy score**
- Simple and efficient pipeline

---

## 🛠️ Tech Stack

- Python 3.x
- Pandas, NumPy
- Scikit-learn
- Plotting

---

## 📊 Dataset

**Pima Indians Diabetes Database**
- 768 samples
- 8 medical features
- Target: Outcome (0 = No Diabetes, 1 = Diabetes)

**Features:**
- Pregnancies
- Glucose
- BloodPressure
- SkinThickness
- Insulin
- BMI
- DiabetesPedigreeFunction
- Age
