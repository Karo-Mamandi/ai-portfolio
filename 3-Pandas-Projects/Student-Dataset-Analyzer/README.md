# Student Dataset Analyzer

A Python program that creates a student dataset and calculates average scores using Pandas.

## Features

- Creates a DataFrame with student information
- Stores student names, ages, and scores
- Calculates the average score of all students
- Demonstrates basic educational data analysis

## How It Works

The program:
1. Creates a Pandas DataFrame with student data
2. Contains columns: Name, Age, Score
3. Includes five students: Karo, Ali, Daniel, Aria, Amir
4. Calculates and displays the average score

## Pandas DataFrame Explained

A DataFrame is a 2-dimensional labeled data structure:
- **Rows**: Individual records (students)
- **Columns**: Features/attributes (Name, Age, Score)

## Requirements
```
pandas>=1.0.0
```
