# Matrix Calculator

A simple Python tool for basic matrix operations using NumPy. Designed as a foundation for larger linear algebra projects.

## Features

- Matrix addition
- Matrix subtraction
- Matrix multiplication
- Determinant calculation

## How It Works

The program:
1. Creates two 2x2 matrices
2. Displays both matrices
3. Performs and displays basic matrix operations:
   - Addition (element-wise)
   - Subtraction (element-wise)
   - Matrix multiplication (dot product)
   - Determinant calculation

## Requirements
```
numpy>=1.15.0
```
