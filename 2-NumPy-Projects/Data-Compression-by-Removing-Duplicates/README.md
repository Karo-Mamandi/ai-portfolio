# Data Compression by Removing Duplicates

A Python program that removes duplicate values from an array and displays the count of each unique value using NumPy.

## How It Works

The program:
1. Generates 30 random integers between 1 and 10
2. Finds all unique values in the array
3. Counts how many times each unique value appears
4. Displays the original data, unique values, and their frequencies

## Requirements
```
numpy>=1.15.0
```
