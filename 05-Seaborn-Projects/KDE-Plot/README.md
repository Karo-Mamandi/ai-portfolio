# KDE Plot with Seaborn

A Python program that creates a KDE (Kernel Density Estimate) plot to visualize the probability density of a numerical variable using Seaborn.

## Features

- Loads the tips dataset from Seaborn
- Creates a KDE plot of total bill amounts
- Shows smooth probability distribution
- Compares distributions by gender

## How It Works

The program:
1. Loads the tips dataset
2. Creates a KDE plot of total_bill
3. Shows the smooth density distribution

## KDE Plot Explained

KDE plot shows probability density:
- Smooth curve instead of bars
- Represents probability distribution
- Great for comparing distributions

## Requirements
```
seaborn>=0.11.0
matplotlib>=3.0.0
```
