# Data Statistics Calculator

A Python program that calculates comprehensive statistical measures from a given dataset using NumPy and Collections.

## Features

- Calculates comprehensive statistics from a fixed dataset:
  - Maximum value
  - Minimum value
  - Mean (average)
  - Median (middle value)
  - Mode (most frequent value)
  - Range (max - min)
  - Variance (spread from mean)
  - Standard Deviation (square root of variance)
  - Coefficient of Variance (relative variability)

## How It Works

The program:
1. Uses a predefined dataset of 20 numbers
2. Displays the dataset
3. Calculates and displays all statistical measures

## Statistical Measures Explained

- **Max**: The largest value in the dataset
- **Min**: The smallest value in the dataset
- **Mean**: The average value (sum ÷ count)
- **Median**: The middle value when data is sorted
- **Mode**: The most frequently occurring value
- **Range**: The difference between maximum and minimum values
- **Variance**: Average of squared differences from the mean
- **Standard Deviation**: Square root of variance (in same units as data)
- **Coefficient of Variance**: (Standard Deviation ÷ Mean) × 100 (percentage)
