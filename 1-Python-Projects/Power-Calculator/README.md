# Power Calculator

A simple Python program that calculates the power of a number using iterative multiplication.

## Description

This project implements a basic power calculation function that takes a base number and an exponent (power) as input, then computes the result by repeatedly multiplying the base with itself. The implementation uses a simple loop approach rather than Python's built-in `**` operator or `pow()` function.

## Features

- Takes user input for base and power values
- Calculates power using iterative multiplication
- Handles positive integer exponents
- Clean and readable code structure
